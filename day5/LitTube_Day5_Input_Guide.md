## 1. STEP 2 — pomelli SNS 캠페인 이미지 생성

### ① pomelli 브랜드 가이드 업로드용 정보

pomelli(pomelli.ai) 에 접속하여 새 브랜드를 생성하고 아래 정보를 입력하세요.

**[브랜드 기본 정보 입력]**

```
브랜드명: LitTube (리튜브)
슬로건: 유튜브를 문학으로 바꾸다
업종: 에듀테크 / 교육 앱
주요 색상: #FF6B35 (오렌지), #1A9396 (테알), #2C333F (차콜), #FDFBF7 (크림 화이트)
보조 색상: #EDE9FE (퍼플 딤), #EAF0E2 (그린 딤)
폰트 스타일: 현대적이고 친근한 산세리프 (Noto Sans KR 계열)
이미지 톤: 따뜻하고 밝은 에듀테크 느낌, 게이미피케이션 요소 포함
로고 설명: 📚 책 이모지 + LitTube 텍스트, 오렌지 컬러 포인트
```

**[캠페인 타겟 설정]**

```
타겟 1 (학생용): 유튜브 숏폼을 즐겨보는 초·중·고등학생, 10~18세, 게임·유튜브·K-pop 관심
타겟 2 (학부모용): 자녀 교육에 관심 많은 30~45세 학부모, 교육 정보·자녀 독서 관련 관심
타겟 3 (교사용): 국어 교사, 방과후 강사, 에듀테크에 열려있는 교육 관계자
```

### ② pomelli 캠페인 생성 프롬프트

pomelli 채팅창에 아래 프롬프트를 입력하여 SNS 광고 이미지를 생성합니다.

**[학생 타겟 인스타그램 캠페인용]**
```
LitTube(리튜브) 에듀테크 앱의 인스타그램 릴스 썸네일 이미지를 만들어줘.

컨셉: "슈카월드 영상이 소설이 된다면?" 
핵심 메시지: 게이머처럼 독서 펫 글몽이를 키우면서 문해력을 키우는 앱
이미지 요소: 
- 귀여운 글몽이 캐릭터 (작고 둥근 흰색 생명체, 책 날개)
- 유튜브 → 소설 변환 화살표 모션감
- "슈카월드 영상이 소설로? 📖" 텍스트 오버레이
색상: 오렌지(#FF6B35) 포인트, 따뜻한 크림 배경
비율: 9:16 (릴스/쇼츠용 세로형)
```

**[학부모 타겟 카카오 채널 배너용]**
```
LitTube(리튜브) 에듀테크 앱의 카카오 채널 광고 배너 이미지를 만들어줘.

컨셉: "아이가 스스로 책을 읽기 시작했어요"
핵심 메시지: 강제 독서 없이 게임처럼 재미있게 문해력을 키우는 솔루션
이미지 요소:
- 따뜻한 조명 아래 책을 읽으며 미소짓는 학생
- 독서 완료 시 글몽이가 춤추는 화면 팝업
- "독서 완료율 85% 달성 🎉" 텍스트 배지
색상: 신뢰감 있는 테알(#1A9396) + 따뜻한 오렌지 포인트
비율: 1:1 또는 16:9 (카카오 채널 배너)
```

## 2. STEP 3 — Gemini Omni Video: 정지 이미지 → 숏폼 영상 변환

### 영상 생성 프롬프트 (Gemini에 이미지와 함께 입력)

Day 4에서 생성한 LitTube 광고 이미지를 Gemini에 첨부한 후 아래 프롬프트를 입력하세요.

**[SCENE 1 — 오프닝: 배고픈 새벽 자취생 → 문제 공감]**
```
이 이미지를 9:16 세로형 숏폼 비디오(7초)로 변환해줘.

카메라 무빙: 정면에서 천천히 줌인 → 학생 얼굴 클로즈업
분위기: 어둡고 무거운 새벽 2시 느낌, 잔잔한 Lo-fi 피아노 배경음
자막 텍스트 (화면 하단): "스마트폰 4.8시간, 그런데 독서는 1.2권"
나레이션 스타일: 잔잔하고 진지한 톤, "이 시간, 당신의 머리를 채울 게 있다면?"
```

**[SCENE 2 — 솔루션: 리튜브 앱 기능 시연]**
```
LitTube 앱 화면 이미지를 9:16 세로형 숏폼 비디오(8초)로 변환해줘.

카메라 무빙: 화면 슬라이드 인(왼쪽에서) → 앱 기능 하이라이트 줌인
효과: 유튜브 URL → 소설 변환 → 글몽이 레벨업 이펙트(골드 파티클) 순서로 전환
분위기: 밝고 에너제틱, 상승하는 비트 음악으로 전환
자막 텍스트: "침착맨 영상이 소설이 됩니다 📚"
나레이션: "즐겨보던 유튜브가 소설이 됩니다. 글몽이가 함께 성장합니다."
```

**[SCENE 3 — CTA: QR + URL 풀스크린]**
```
오렌지 배경의 LitTube 브랜드 이미지를 9:16 세로형 숏폼 비디오(5초)로 변환해줘.

카메라 무빙: 정적인 풀스크린 유지 (카메라 무빙 없음)
효과: QR코드와 URL이 화면 중앙에 자연스럽게 페이드 인
자막 텍스트 (상단): "지금 무료로 글몽이를 만나세요! 🐾"
자막 텍스트 (하단): "📚 littube.glittersuhodae.netlify.app"
나레이션: 밝고 활기찬 목소리 "사전 체험 신청은 링크에서!"
```

## 3. STEP 4 — Google Vids 최종 영상 빌딩

### Google Vids 설정 및 AI 기능 입력값

vids.google.com 에서 새 프로젝트를 만들고 아래 순서대로 진행하세요.

**[기본 설정]**
```
비율: 9:16 (세로형 숏폼)
총 길이: 20초
제목: LitTube_론칭_홍보영상_v1
```

**[AI 배경음악 생성 프롬프트]**
```
SCENE 1 (0~7초): 잔잔하고 무거운 Lo-fi 피아노. BPM 70. 약간 슬프고 고요한 분위기.
SCENE 2 (7~15초): 상승하는 에너제틱 비트로 급전환. BPM 120. 신나고 희망찬 에듀테크 느낌.
SCENE 3 (15~20초): 밝고 완결된 느낌의 짧은 마무리 음악. 브랜드 징글 효과.
장르: Lo-fi + Upbeat Pop, 저작권 무료
```

**[AI 성우 나레이션 원고 — 전체 20초]**

```
[SCENE 1 나레이션 원고 — 0:00~0:07, 잔잔하고 진지한 톤]
"스마트폰 4.8시간... 하지만 책은 1.2권."
"(짧은 침묵)"
"이 시간, 당신의 아이 머리를 채울 수 있는 게 있다면?"

[SCENE 2 나레이션 원고 — 0:07~0:15, 밝고 활기차게 전환]
"즐겨보던 유튜브가 소설이 됩니다."
"독서 펫 글몽이가 함께 성장합니다."
"게임처럼 시작해서, 문학으로 끝납니다."

[SCENE 3 나레이션 원고 — 0:15~0:20, 빠르고 신나게]
"지금 무료로 글몽이를 만나세요!"
```

**AI 성우 설정:**
```
성별: 여성
연령: 20대 후반
톤: SCENE 1은 차분하고 진지하게, SCENE 2~3은 밝고 신뢰감 있게
속도: 보통 (단, SCENE 3은 빠르게)
문장 부호 처리: 마침표에서 0.5초 호흡, 생략 부호(...)에서 1초 호흡
```

**[자막 및 텍스트 오버레이 추가]**

| 시간 | 자막 텍스트 | 위치 | 폰트 스타일 |
| --- | --- | --- | --- |
| 0:00~0:07 | "스마트폰 4.8h vs 독서 1.2권" | 화면 하단 중앙 | 흰색, 굵게 |
| 0:07~0:15 | "침착맨 영상 → 소설 변환 📚" | 화면 상단 | 오렌지, 굵게 |
| 0:15~0:20 | "🐾 littube.glittersuhodae.netlify.app" | 화면 하단 전체 | 흰색 + QR코드 |
