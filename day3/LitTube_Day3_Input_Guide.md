## 2. STEP 3 — Few-shot 예시뱅크 (FEWSHOT 단계 입력값)

AI에게 문장 스타일을 가르치기 위한 **나쁜 예 → 좋은 예 3쌍**입니다.  
실습도구 FEWSHOT 단계에 아래 내용을 입력하고, Gemini에 함께 붙여넣어 주세요.

### Few-shot 예시 3쌍

**[쌍 1] 제품 설명 문장**
```
❌ 나쁜 예: "리튜브는 유튜브와 독서를 연결하는 좋은 서비스입니다. 앞으로 많이 발전할 것입니다."

✅ 좋은 예: "LitTube는 학생이 즐겨보는 유튜브 영상의 URL을 입력하면 AI가 3초 내에 초·중·고 맞춤 소설·시·수필로 변환한다. 파일럿 테스트에서 독서 완료율이 기존 대비 4.2배(20%→85%) 향상됐다."
```

**[쌍 2] 타겟 고객 설명 문장**
```
❌ 나쁜 예: "우리 타겟은 공부를 싫어하는 학생들입니다. 학습 의욕이 없어서 책을 안 읽습니다."

✅ 좋은 예: "타겟은 하루 4.8시간 유튜브를 보지만 연간 독서량은 1.2권인 알파세대 청소년이다. 이들은 책이 싫은 게 아니라, '재미없는 소재'가 싫다 — 자신이 구독하는 유튜버의 영상이 소설이 된다면 거부감보다 호기심이 앞선다."
```

**[쌍 3] 성과 주장 문장**
```
❌ 나쁜 예: "리튜브를 쓰면 문해력이 많이 좋아집니다. 아이들이 책을 더 많이 읽게 됩니다."

✅ 좋은 예: "LitTube 파일럿 테스트(학급 5곳, 4주간)에서 주당 독서 완료율이 20%에서 85%로 급등했고, 단어 암기 지속률은 76%로 기존 암기식 학습(18%) 대비 4배 이상 향상됐다."
```

## 3. STEP 4~5 — 8장 기획서 본문 (WRITE 단계 입력값)

### Gemini에 붙여넣을 기획서 생성 최종 프롬프트

```text
[Goal]
아래 ISI 골격과 Few-shot 예시를 바탕으로, 투자자·교육청 담당자를 설득할 수 있는
LitTube(리튜브) 에듀테크 서비스의 8장 브랜드 기획서를 작성해줘.
각 페이지는 두괄식(결론 → 근거 → 데이터 → 다음 액션) 구조를 철저히 따라줘.

[Few-shot — 이 스타일로 써줘]
❌ "리튜브는 좋은 서비스입니다." 
✅ "[시장 기회] 3,500억 문해력 에듀테크 시장, 연 12.5% 성장. 영상+독서 융합 솔루션은 현재 공백 영역."

❌ "학생들이 공부를 싫어해서 문제입니다."
✅ "'책이 싫은 게 아니라, 재미없는 책이 싫다.' 자기가 좋아하는 유튜버의 영상이 소설이 된다면, 거부감보다 호기심이 앞선다."

[ISI 골격]
- Insight: 알파세대 청소년 스마트폰 4.8시간 vs 독서 1.2권. 강제 독서는 역효과. 유튜브를 문학으로 바꾸는 것이 해법.
- Strategy: 유튜브 URL → AI 소설/시/수필 3종 변환 + 독서 펫 '글몽이' 게이미피케이션.
- Impact: 6개월 내 100개 학급, 독서 완료율 85%, 연 ARR 3.6억.

[8장 구조]
P1: 문제 정의 / P2: 시장 기회 / P3: 소비자 인사이트 / P4: 제품 컨셉
P5: 마케팅 전략 / P6: 실행 일정 / P7: 예상 성과 / P8: 결론 및 투자 제안

[Context]
- 서비스명: LitTube(리튜브)
- 타겟: 초·중·고등학생 / 학부모(B2C) / 학교 교사(B2B) / 교육청(B2G)
- 핵심 수치: 에듀테크 시장 8.2조, 문해력 세그먼트 3,500억, 연 12.5% 성장, 독서 완료율 85%, 초기 투자 1,800만원

[Role]
너는 스타트업 엑셀러레이터 출신의 시니어 사업기획자야.
투자자의 마음을 여는 스토리텔링 기획서를 스티브 잡스처럼 단순하고 강렬하게 써줘.
```

### 8장 기획서 최종 완성 내용 (직접 입력용)

| 페이지 | 제목 | 핵심 내용 |
| :--- | :--- | :--- |
| **P1** | 문제 정의 | 청소년 일평균 스마트폰 사용 4.8시간, 연간 독서 1.2권. 교과서 지문을 반 이상 이해 못 하는 '문맹 이상, 독해 불능' 세대가 출현하고 있다. 강제 독서 처방은 이탈율 80%로 역효과. |
| **P2** | 시장 기회 | 국내 에듀테크 시장 8.2조 원, 문해력·리터러시 세그먼트 3,500억 원, 연 12.5% 성장. 영상+독서 융합 솔루션은 현재 시장에 단 하나도 없는 공백 영역. |
| **P3** | 소비자 인사이트 | "책이 싫은 게 아니라, 재미없는 책이 싫다." 자신이 좋아하는 유튜버의 영상이 소설이 된다면, 거부감보다 호기심이 먼저 온다. (중학교 2학년 포커스그룹 인터뷰 인용) |
| **P4** | 제품 컨셉 | 유튜브 URL 입력 → Gemini 2.5 스크립트 분석 → 초중고 맞춤 소설·시·수필 3종 변환 → 스마트 사전 + 퀴즈 + 독서 펫 '글몽이' 레벨업 게이미피케이션 전체 연동. |
| **P5** | 마케팅 전략 | ①슈카월드·과학쿠키 등 교육 유튜버 콜라보 바이럴(B2C) ②방과후 수업 연계 학교 구독(B2B) ③교육부 디지털 교과서 파일럿 납품(B2G) 3단계 GTM 전략. |
| **P6** | 실행 일정 | M1~M2: MVP 앱 완성 + 학교 5곳 파일럿. M3~M6: 교사 얼리어답터 1,000명 온보딩. M7~M12: 전국 100개 학급 정식 도입. |
| **P7** | 예상 성과 | 6개월 말 MAU 5만 명, 독서 완료율 85%, 단어 습득 4.2배. 학교 구독 월 30,000원 × 100개 학급 = 연 3.6억 ARR 달성. |
| **P8** | 결론 & 투자 제안 | 초기 개발비 1,800만 원(AI API, UI 개발, 서버). 교육부 에듀테크 진흥 보조금 신청 동시 추진. 12개월 내 SaaS 전환 및 Series A 준비. |
