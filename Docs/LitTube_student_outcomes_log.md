# LitTube(리튜브) 5일 수업 결과물 샘플 구현 로그

* **프로젝트**: LitTube (리튜브) — 유튜브 영상 기반 AI 문해력 증진 에듀테크
* **기록일자**: 2026-06-28
* **작성자**: 서기 (Scribe)
* **목적**: Day 1~5 각 단계별 최종 수업 결과물 샘플 및 AI 최종 프롬프트 제작 내역 기록

---

## 1. 구현 개요 (Implementation Summary)

LitTube(리튜브) 기획안을 주제로, 5일 AI 창업 수업 과정을 완주했을 때 단계별로 산출되는 **실제 최종 결과물 5종**을 단독으로 체험 가능한 파일 형태로 제작 완료함.

각 파일은 단순 텍스트 보고서에 그치지 않고, **브라우저에서 즉시 실행 가능한 인터랙티브 HTML 형태**로 제작되어 실제 학생들이 5일 과정을 마쳤을 때 제출하는 산출물처럼 작동하며 체험할 수 있도록 구성함.

---

## 2. 일차별 산출물 및 구현 내용 (Deliverables per Day)

### DAY 1 — 팀 미션 키트 & AI OKR 프롬프트
- **파일**: `day1/LitTube_Day1_Concept.md`
- **내용**: 팀명(글터수호대), 타겟, 문제 정의, 컨셉 시드 4칸으로 구성된 팀 미션 키트.
- **최종 프롬프트**: Gemini에 바로 붙여넣어 실행 가능한 **AI OKR 4칸 조립 프롬프트** (Goal/Context/Sample/Role).
- **AI 피드백 요약**: 유튜브와 문학 독서를 결합한 게이미피케이션 에듀테크 컨셉의 차별성, 경쟁사 분석 표 제공.

### DAY 2 — 시장 조사 인사이트 보고서 & 인터랙티브 차트 3종
- **파일**: `day2/LitTube_Day2_Insight_Report.html`
- **주요 구현 내용**:
  - 두괄식 헤드라인 및 2페이지 분량의 인사이트 보고서 본문 (에듀테크 시장 8.2조, 문해력 세그먼트 3,500억 원, 연 12.5% 성장 데이터).
  - **SVG 네이티브 차트 3종** 구현:
    - 라인 차트: 스마트폰 사용 시간 상승 vs 독서량 하락 추이 (Draw 애니메이션).
    - 도넛 차트: 청소년 문해력 저하 원인 비중 (Stroke-dasharray 애니메이션).
    - 막대 차트: 기존 독서 vs 리튜브 게이미피케이션 독서 완료율 비교 (ScaleY 애니메이션).
  - **최종 프롬프트**: NotebookLM 딥리서치 및 KSA 시장 조사 보고서 생성을 위한 조립 프롬프트.

### DAY 3 — 8장 ISI 기획서 & 인터랙티브 발표 슬라이드 덱
- **파일**: `day3/LitTube_Day3_ISI_Proposal.html`
- **주요 구현 내용**:
  - ISI (Insight→Strategy→Impact) 3단 골격 배너 및 8장 기획서(P1~P8) 벤토 그리드.
  - **슬라이드쇼 모드**: "슬라이드 보기" 버튼 클릭 시 전환. 좌우 방향키 및 클릭 버튼으로 6장의 발표 슬라이드 전환 가능.
  - 각 슬라이드 하단에 실제 발표 대본 (Script) 실시간 연동.
  - **최종 프롬프트**: Few-shot 예시(좋은 예/나쁜 예)를 포함한 8장 기획서 생성 프롬프트.

### DAY 4 — 리튜브 앱 핵심 기능 프로토타입 랜딩 페이지
- **파일**: `day4/LitTube_Day4_LandingPage.html`
- **주요 구현 내용**:
  - 전체 랜딩 페이지 레이아웃 (Hero, How it works, Stats, CTA).
  - **핵심 인터랙티브 데모**:
    - 유튜브 URL 입력 시 분석 바(Loading Bar) + 단계별 상태 메시지 애니메이션.
    - 소설/시/수필 3종 장르 선택기 및 독서실(Reading Room) 진입.
    - 테마 스위처 (종이/다크/크림) 및 폰트 크기 조절 툴바.
    - 스크롤 진행도 바 및 글몽이 이모지 반응 (💤→🏃→🎉).
    - 본문 내 하이라이트 단어 클릭 시 뜻풀이 툴팁 팝업 + 단어장 수집 기능.
    - 독서 완료(100% 스크롤) 시 퀴즈 자동 등장 및 정답 판정 + EXP/골드 보상.
    - 로컬 스토리지(Local Storage)를 통해 레벨, EXP, 수집 단어 보존.
  - **최종 프롬프트**: Gemini Gems 브랜드 매니저 '글튜브 AI' 생성 지침 + Google Stitch 5섹션 랜딩 페이지 빌딩 프롬프트.

### DAY 5 — 숏폼 홍보 영상 시뮬레이터 & 5단 론칭 피칭 대본
- **파일**: `day5/LitTube_Day5_Video_Pitch.html`
- **주요 구현 내용**:
  - **세로형(9:16) 폰 프레임 내 영상 플레이어 시뮬레이터**:
    - ▶ 재생 버튼 클릭 시 1초 간격 타이머로 Scene 1→2→3 자동 전환 (opacity 트랜지션).
    - 하단 진행 바 및 타임코드 실시간 업데이트.
    - 타임라인 도트(●) 클릭 시 특정 Scene으로 점프.
    - 재생 완료 후 "다시 재생" 버튼 노출.
  - 각 Scene 상세 텍스트 카드 (나레이션 원고 포함).
  - 5단 론칭 피칭 대본 카드 (오프닝 후크/문제/솔루션/영상/CTA+투자제안).
  - **최종 프롬프트**: Gemini Omni Video 숏폼 영상 생성 프롬프트 + Google Vids AI 성우 나레이션 및 피칭 대본 생성 프롬프트.

## 4. 기술 스택 및 디자인 시스템 (Tech Stack)

- **CSS/UI 프레임워크**: Tailwind CSS CDN (V3.4), Font Awesome 6.4.0
- **폰트**: Google Fonts (Noto Sans KR, Outfit, JetBrains Mono, Nanum Myeongjo)
- **차트**: 외부 라이브러리 없이 네이티브 SVG + CSS 애니메이션 (`stroke-dasharray`, `transform: scaleY`, `stroke-dashoffset`)
- **데이터 영속성**: `localStorage` (레벨, EXP, 단어장)
- **디자인 컨셉**: Glassmorphism 2.0 (반투명 카드, backdrop-blur), 웜 브라이트 테마 (#FDFBF7 배경), 오렌지 포인트 컬러 (#FF6B35), 벤토 그리드 레이아웃
- **인터랙션**: CSS 키프레임 애니메이션, scroll 이벤트 기반 진행도 추적, 클립보드 API (프롬프트 복사)
- **공통 링크**: 모든 파일 하단 AK LABS 홈페이지 링크 삽입 (https://litt.ly/aklabs)
