## 2. 세부 변경 사항
### A. STEP 4 (구글 비즈 오디오 & 나레이션) 입력 폼 제거
- 기존에 불필요한 중복 텍스트 입력을 강요했던 `BGM 묘사 입력창`, `배포 URL 입력창`, `TTS 나레이션 대본 입력창`을 HTML에서 전면 삭제.
- 대신 학생들이 직접 Vids 창에 붙여넣어 활용할 수 있는 **'추천 AI BGM 프롬프트 복사 카드'** 3가지(Lofi 피아노, 미니멀 앰비언트, 어쿠스틱 기타)를 원클릭 복사 칩 형태로 도입.
- AI 성우가 자연스럽게 호흡을 조절하며 나레이션을 읽을 수 있도록 돕는 **'쉼표(,) & 온점(.) 가이드라인'** 텍스트를 배치하고, 표준 쉼표 가드 대본 예시 복사 버튼을 추가.

### B. 자바스크립트 런타임 Null 에러 예방
- HTML에서 삭제된 인풋 요소들(`inp-vids-bgm`, `inp-vids-url`, `inp-vids-tts`)을 DOM에서 조회해 `.value`를 대입하던 자바스크립트 `assemblePrompt()` 함수 내 로직을 정리.
- 각 값에 기본 정적 문자열 값을 고정 할당하도록 변경하여 브라우저 개발자 도구 콘솔의 `TypeError`를 완전히 사전에 차단함.
- `vids-bgm-preview`, `vids-tts-preview` 등 삭제된 돔 영역에 강제로 대입하려던 라인들 제거.
